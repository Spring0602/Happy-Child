# 01｜《快乐小孩》像素美术风格规范 v0.2

> 用途：统一 AI 生成像素场景、角色动作、UI、特效、CG 的视觉方向。带 `[待填]` 的位置后续补充。

## 1. 项目视觉定位

| 项目 | 内容 |
|---|---|
| 游戏类型 | 2D像素风AI叙事视觉小说 |
| 核心体验 | 玩家探索地图，触发剧情时显示对话框、立绘、AI分析 |
| 主题关键词 | 规训、好孩子、快乐、反抗、主体性、镜中真相 |
| 情绪基调 | 压抑、冷色、窒息、偶尔出现温柔和希望 |
| 像素风格参考 | 早期GBA宝可梦 / 尸体派对（低分辨率，暗色调） |

## 2. 像素规格

| 项目 | 规格 | 状态 |
|---|---:|---|
| 地图 tile | 32×32 px | [待填] |
| **角色精灵图** | **256×256 px 单张 Sprite Sheet（含所有动作）** | ✅ 已确认 |
| 每帧内容尺寸 | 约 17×40 px | ✅ 已确认 |
| 帧布局 | 行0: 站立(3帧), 行1: 跑步(4帧), 行2: 攻击(4帧), 行3: 受击(3-4帧) | ✅ 已确认 |
| 行间距 | 约 24 px | ✅ 已确认 |
| 帧间距 | 约 25 px | ✅ 已确认 |
| 颜色数 | ≤12 色 | ✅ 已确认 |
| 轮廓 | 1px 黑色描边 | ✅ 已确认 |
| 阴影 | 2层色阶（底色+暗部，无高光） | ✅ 已确认 |
| 游戏画面 | 1280×720，16:9 | [待填] |
| 立绘 | 透明PNG，高度约1200px | [待填] |
| CG | 1280×720 或 1920×1080 | [待填] |

角色精灵图布局（单张256×256 px）：

```text

└─────────────────────────────────────────────┘
```

> 右侧面 = 左侧面代码翻转，不单独占行。
