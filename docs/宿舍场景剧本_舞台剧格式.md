## 舞台说明

**布景**：一间四人大学男生宿舍。四张上下铺床分列左右两侧。房间中央靠墙是一排书桌。每张书桌上堆满了课本、笔记本电脑和杂物。窗户在左侧墙壁，阳台门在窗户旁。宿舍门在画面底部偏右。

**灯光**：凌晨的暗蓝色调。房间整体昏暗，仅有窗外路灯的微光透入。主角书桌上的笔记本电脑屏幕是唯一的光源，散发着幽幽的白光，照亮主角的脸。

**音效**：窗外淅淅沥沥的雨声。偶尔远处有卡车驶过的声音。键盘敲击声（开场时）。时钟秒针走动声。

**时间**：凌晨 04:46。

## 技术实现说明

### flag 状态机

```
阳台回来 → dorm_act2_think 关闭 → SET_FLAG dorm_act2
    ↓
自由探索（act2 拦截生效）
    ↓
交互电脑 → dorm_act2_pc_confirm
    ├── 继续探索 → dorm_act2_explore 关闭 → SET_FLAG dorm_act2_exploring
    │     ↓
    │   自由探索（act2_exploring 拦截生效）
    │     ↓
    │   再次交互电脑 → dorm_act2_pc_force_sleep → dorm_act2_sleep_result
    │
    └── 睡觉 → dorm_act2_sleep_result
```

### PhaserContainer 闭包修复

`PhaserContainer` 的 `useEffect` 依赖为 `[]`，仅挂载时运行一次。但 `handleDialogueTrigger` 通过 `useCallback` 依赖 `state.flags`。当 flags 更新时，`handleDialogueTrigger` 重新创建，但 gameBridge 回调闭包中捕获的仍是旧版本。

**修复**：使用 `useRef` 保存 `onDialogueTrigger` 和 `dispatch` 最新引用：
```typescript
const onDialogueTriggerRef = useRef(onDialogueTrigger);
const dispatchRef = useRef(dispatch);
onDialogueTriggerRef.current = onDialogueTrigger;
dispatchRef.current = dispatch;
// 回调中使用 onDialogueTriggerRef.current(event.sceneId)
```
