## 1. 架构设计

### 1.1 融合架构（探索 + 叙事合一）

探索和叙事不再互斥切换，而是**融合在一起**：

- **Phaser Canvas 始终可见** — 玩家始终能看到当前所在区域的像素地图
- **对话以浮动叠层呈现** — 触发剧情时，底部弹出紧凑对话框（不遮挡场景）
- **共用 GameState** — Phaser 和 React 共享同一份状态

| 层级 | 渲染 | 交互 | 说明 |
|------|------|------|------|
| **底层层** | Phaser Canvas | WASD移动 + E交互 + 碰撞 | 始终渲染，对话时冻结移动 |
| **叠层层** | React DialogOverlay | 点击选项/继续/AI分析 | 有对话时显示，关闭后隐藏 |

```
App.tsx
├── GameState (useReducer, 共享)
├── GameBridge (事件总线)
├── <PhaserContainer /> (始终渲染)
├── <DialogOverlay /> (有对话时叠加)
├── <StatusPanel /> (7维人格, 始终可见)
└── <AITracePanel /> (AI输出, 始终可见)
```

### 1.2 对话触发流程

```
探索中(Phaser)
    │
    │  玩家走到触发点/NPC旁按E
    │  ────── GameBridge ──────→
    │  {TRIGGER_DIALOGUE, sceneId}
    │
    ├── FREEZE_PLAYER（冻结移动）
    ├── DIALOG_START（React显示DialogOverlay）
    │
    │  ... 对话推进中（点击继续/选择/AI分析）...
    │
    ├── 对话链结束（无 nextSceneId / choices / aiEvent）
    ├── DIALOG_END（隐藏DialogOverlay）
    ├── UNFREEZE_PLAYER（恢复移动）
    │
    ▼ 玩家继续探索

  | { type: "DIALOG_END" };

Reducer 中新增 case：

```typescript
case "CHANGE_MAP":
  return { ...state, currentMapId: action.mapId, playerPosition: action.position };

case "SET_FLAG":
  return { ...state, flags: { ...state.flags, [action.flag]: true } };

case "INTERACT_ITEM":
  return { ...state, interactedItems: [...state.interactedItems, action.itemId] };

case "UPDATE_POSITION":
  return { ...state, playerPosition: action.position };

case "DIALOG_START":
  return { ...state, currentSceneId: action.sceneId };

case "DIALOG_END":
  return { ...state, currentSceneId: "" };
```

> **注意**：不再有 `SWITCH_MODE` action。对话不再通过"切换模式"触发，而是通过 `DIALOG_START` 设置 `currentSceneId`，React 检测到有效 sceneId 后显示 DialogOverlay 叠层。`DIALOG_END` 将 `currentSceneId` 置空以隐藏叠层。

## 5. GameBridge 实现

### `client/src/game/bridge/GameBridge.ts`

```typescript
import type { PhaserToReactEvent, ReactToPhaserCommand } from "../types/gameMap";
import type { GameState } from "../../types/game";

type Listener<T> = (event: T) => void;

/**
 * Phaser ↔ React 通信桥（发布-订阅模式）
 * 
 * 使用方式：
 * - React 侧：gameBridge.onPhaserEvent(...) 监听 Phaser 事件
 * - Phaser 侧：gameBridge.onReactCommand(...) 监听 React 指令
 * - 跨引擎通信：sendToReact / sendToPhaser
 */
class GameBridge {
  private phaserListeners = new Map<string, Listener<ReactToPhaserCommand>[]>();
  private reactListeners = new Map<string, Listener<PhaserToReactEvent>[]>();
  private _gameState: GameState | null = null;

  // Phaser 侧调用：监听 React 发来的指令
  onReactCommand(type: string, handler: Listener<ReactToPhaserCommand>) {
    if (!this.phaserListeners.has(type)) this.phaserListeners.set(type, []);
    this.phaserListeners.get(type)!.push(handler);
  }

  // React 侧调用：监听 Phaser 发来的事件
  onPhaserEvent(type: string, handler: Listener<PhaserToReactEvent>) {
    if (!this.reactListeners.has(type)) this.reactListeners.set(type, []);
    this.reactListeners.get(type)!.push(handler);
  }

  // React → Phaser
  sendToPhaser(command: ReactToPhaserCommand) {
    const handlers = this.phaserListeners.get(command.type) || [];
    handlers.forEach((h) => h(command));
  }

  // Phaser → React
  sendToReact(event: PhaserToReactEvent) {
    const handlers = this.reactListeners.get(event.type) || [];
    handlers.forEach((h) => h(event));
  }

  // 共享 GameState
  set gameState(state: GameState | null) { this._gameState = state; }
  get gameState() { return this._gameState; }

  // 清理
  removeAllListeners() {
    this.phaserListeners.clear();
    this.reactListeners.clear();
  }
}

export const gameBridge = new GameBridge();
```

### 6.2 PreloadScene

> 本项目使用 **Image Layer** 方案（每张地图以整张底图作为地面），PreloadScene 加载地图 JSON、底图、家具图片和角色精灵表。

```typescript
// client/src/game/scenes/PreloadScene.ts
import Phaser from "phaser";
import { AssetManifest } from "../config/assetManifest";
import { preloadSpriteSheets, createPlayerAnimations } from "../config/animationRegistry";

export class PreloadScene extends Phaser.Scene {
  constructor() { super({ key: "PreloadScene" }); }

  preload() {
    // 加载地图 JSON + 底图（Image Layer 方案）
    for (const map of Object.values(AssetManifest.maps)) {
      this.load.tilemapTiledJSON(map.mapKey, map.mapJson);
      this.load.image(map.tilesetKey, map.tilesetImage);
      if (map.groundImage) this.load.image(map.groundKey!, map.groundImage);
    }

    // 加载家具独立图片
    for (const map of Object.values(AssetManifest.maps)) {
      if (map.furnitureImages) {
        for (const img of map.furnitureImages) {
          this.load.image(img.key, img.path);
        }
      }
    }

    // 加载角色动作 spritesheet（由 GIF 预转的 PNG）
    // 每张 PNG = 192×64px，6帧横排，frameWidth=32, frameHeight=64
    preloadSpriteSheets(this);
  }

  create() {
    createPlayerAnimations(this);
    this.scene.start("MapScene", { mapId: "livingroom" });
  }
}

### 6.3 MapScene（核心场景）

```typescript
// client/src/game/scenes/MapScene.ts
import Phaser from "phaser";
import { MapRegistry } from "../config/mapRegistry";
import { gameBridge } from "../bridge/GameBridge";
import { PlayerController } from "../systems/PlayerController";
import { InteractionSystem } from "../systems/InteractionSystem";
import { TriggerSystem } from "../systems/TriggerSystem";

export class MapScene extends Phaser.Scene {
  private map!: Phaser.Tilemaps.Tilemap;
  private player!: Phaser.GameObjects.Sprite;
  private playerCtrl!: PlayerController;
  private interactionSys!: InteractionSystem;
  private triggerSys!: TriggerSystem;
  private collisionGroup!: Phaser.Physics.Arcade.StaticGroup;
  private currentMapId = "";

  constructor() { super({ key: "MapScene" }); }

  init(data: { mapId: string }) {
    this.currentMapId = data.mapId;
  }

  create() {
    this.loadMap(this.currentMapId);
    this.listenBridge();
  }

  /** 加载一张地图 */
  private loadMap(mapId: string, spawnId?: string) {
    const entry = MapRegistry[mapId];
    if (!entry) { console.error(`地图 ${mapId} 未注册`); return; }

    // 清理旧场景对象
    this.children.removeAll();

    // 创建 tilemap
    this.map = this.make.tilemap({ key: entry.mapKey });
    const tileset = this.map.addTilesetImage(
      entry.tilesetNameInTiled, entry.tilesetKey,
      entry.tileWidth, entry.tileHeight
    );

    // 渲染 tile 图层
    const groundLayer = this.map.createLayer("ground", tileset, 0, 0);
    if (groundLayer) {
      groundLayer.setCollisionByExclusion([2]); // tile 2=地板, 不碰撞
    }
    this.map.createLayer("furniture", tileset, 0, 0);

    // 碰撞组
    this.collisionGroup = this.physics.add.staticGroup();

    // 处理对象层
    this.processObjectLayers(spawnId || entry.defaultSpawn);

    // 加载家具独立图片（放在 tile 图层之上）
    if (entry.furnitureImages) {
      const furnitureObjLayer = this.map.getObjectLayer("furniture_objects");
      if (furnitureObjLayer) {
        for (const obj of furnitureObjLayer.objects) {
          const match = entry.furnitureImages.find((f) => f.key === obj.name);
          if (match) {
            const sprite = this.add.image(
              obj.x! + (obj.width ?? 0) / 2,
              obj.y! + (obj.height ?? 0) / 2,
              match.key
            );
            // Y 排序：y 坐标越大越靠前
            sprite.setDepth((obj.y ?? 0) / this.map.heightInPixels);
            // 碰撞盒
            const body = this.physics.add.staticBody(
              obj.x!, obj.y!, obj.width ?? 64, obj.height ?? 64
            );
            this.collisionGroup.add(body);
          }
        }
      }
    }

    // 创建玩家
    this.createPlayer(spawnId || entry.defaultSpawn);

    // 设置世界边界 & 相机
    this.physics.world.setBounds(0, 0, this.map.widthInPixels, this.map.heightInPixels);
    this.cameras.main.setBounds(0, 0, this.map.widthInPixels, this.map.heightInPixels);
    // 大地图用更缓的 lerp，小场景即时跟随
    const lerp = this.map.widthInPixels > 2000 ? 0.05 : 0.08;
    this.cameras.main.startFollow(this.player, true, lerp, lerp);

    // 初始化子系统
    this.playerCtrl = new PlayerController(this, this.player);
    this.interactionSys = new InteractionSystem(this, this.player);
    this.triggerSys = new TriggerSystem(this, this.player);
  }

  /** 处理 Tiled 对象层 */
  private processObjectLayers(defaultSpawn: string) {
    // --- collision ---
    const collisionLayer = this.map.getObjectLayer("collision");
    if (collisionLayer) {
      for (const obj of collisionLayer.objects) {
        const body = this.physics.add.staticBody(
          obj.x!, obj.y!, obj.width ?? 32, obj.height ?? 32
        );
        this.collisionGroup.add(body);
      }
    }

    // --- npcs ---
    const npcLayer = this.map.getObjectLayer("npcs");
    if (npcLayer) {
      for (const obj of npcLayer.objects) {
        const props = this.getProps(obj.properties);
        // 占位矩形（后续替换为真实精灵图）
        const rect = this.add.rectangle(
          obj.x! + (obj.width ?? 32) / 2,
          obj.y! + (obj.height ?? 32) / 2,
          obj.width ?? 32, obj.height ?? 32,
          0x4488ff, 0.6
        );
        rect.setDepth((obj.y ?? 0) / this.map.heightInPixels);
        // NPC 碰撞体
        const body = this.physics.add.staticBody(
          obj.x!, obj.y!, obj.width ?? 32, obj.height ?? 32
        );
        this.collisionGroup.add(body);
        // 注册为可交互对象
        this.interactionSys?.registerInteractable({
          x: obj.x! + (obj.width ?? 32) / 2,
          y: obj.y! + (obj.height ?? 32) / 2,
          type: "npc",
          id: props.npcId || obj.name,
          sceneId: props.sceneId,
        });
      }
    }

    // --- triggers ---
    const triggerLayer = this.map.getObjectLayer("triggers");
    if (triggerLayer) {
      for (const obj of triggerLayer.objects) {
        const props = this.getProps(obj.properties);
        const zone = this.add.zone(
          obj.x! + (obj.width ?? 64) / 2,
          obj.y! + (obj.height ?? 64) / 2,
          obj.width ?? 64, obj.height ?? 64
        );
        this.physics.add.existing(zone, true);
        this.triggerSys?.registerTrigger(zone as any, {
          type: props.type || "dialogue",
          sceneId: props.sceneId,
          targetMap: props.targetMap,
          spawnId: props.spawnId,
          requireFlag: props.requireFlag,
          setFlag: props.setFlag,
        });
      }
    }

    // --- items ---
    const itemsLayer = this.map.getObjectLayer("items");
    if (itemsLayer) {
      for (const obj of itemsLayer.objects) {
        const props = this.getProps(obj.properties);
        const rect = this.add.rectangle(
          obj.x! + (obj.width ?? 24) / 2,
          obj.y! + (obj.height ?? 24) / 2,
          obj.width ?? 24, obj.height ?? 24,
          0xffaa00, 0.6
        );
        this.interactionSys?.registerInteractable({
          x: obj.x! + (obj.width ?? 24) / 2,
          y: obj.y! + (obj.height ?? 24) / 2,
          type: "item",
          id: props.itemId || obj.name,
          sceneId: props.sceneId,
        });
      }
    }
  }

  /** 创建玩家 Sprite */
  private createPlayer(spawnId: string) {
    const spawnLayer = this.map.getObjectLayer("player_spawn");
    let sx = this.map.widthInPixels / 2;
    let sy = this.map.heightInPixels / 2;
    if (spawnLayer) {
      const spawn = spawnLayer.objects.find((o) => o.name === spawnId);
      if (spawn) {
        sx = spawn.x! + (spawn.width ?? 32) / 2;
        sy = spawn.y! + (spawn.height ?? 32) / 2;
      }
    }

    this.player = this.add.sprite(sx, sy, "yps_front");  // 使用 front 方向 spritesheet
    this.physics.add.existing(this.player);
    const body = this.player.body as Phaser.Physics.Arcade.Body;
    body.setCollideWorldBounds(true);
    body.setSize(17, 40);
    body.setOffset(8, 16);

    // 初始待机动画（默认朝下/正面）
    this.player.play("yps_idle_front");

    // 碰撞：玩家 vs ground tile 碰撞
    const groundLayer = this.map.getLayer("ground")?.tilemapLayer;
    if (groundLayer) this.physics.add.collider(this.player, groundLayer);
    // 碰撞：玩家 vs 碰撞组
    this.physics.add.collider(this.player, this.collisionGroup);
  }

  /** 监听 GameBridge 指令 */
  private listenBridge() {
    gameBridge.onReactCommand("CHANGE_MAP", (cmd) => {
      if (cmd.type === "CHANGE_MAP") this.transitionToMap(cmd.mapId, cmd.spawnId);
    });
    gameBridge.onReactCommand("FREEZE_PLAYER", () => this.playerCtrl?.freeze());
    gameBridge.onReactCommand("UNFREEZE_PLAYER", () => this.playerCtrl?.unfreeze());
  }

  /** 地图切换（淡入淡出） */
  private transitionToMap(mapId: string, spawnId: string) {
    this.cameras.main.fadeOut(300, 0, 0, 0);
    this.cameras.main.once("camerafadeoutcomplete", () => {
      this.loadMap(mapId, spawnId);
      this.cameras.main.fadeIn(300, 0, 0, 0);
    });
  }

  /** 从 Tiled properties 数组中提取值 */
  private getProps(properties?: { name: string; value: any }[]): Record<string, string> {
    const result: Record<string, string> = {};
    if (!properties) return result;
    for (const p of properties) { result[p.name] = String(p.value); }
    return result;
  }

  update(_time: number, delta: number) {
    this.playerCtrl?.update(delta);
    this.interactionSys?.update();
    this.triggerSys?.update();
    // Y 排序
    this.player.setDepth(this.player.y / this.map.heightInPixels);
  }
}
```

## 7. 子系统实现

### 7.1 PlayerController

> **动画机制（v8.0 / GIF序列帧方案）**：WASD 控制四方向移动。每个方向的跑步动画为独立 GIF 文件（`_front`/`_back`/`_left`/`_right`），单帧尺寸 32×64px，每个 GIF 共 6 帧。Phaser 通过 `phaser3-rex-plugins` 或直接用 `GIFDecoder` 将 GIF 分解为 spritesheet，再用 `anims.create` 注册动画。右朝向直接加载 `_right` GIF（已预生成镜像），无需 setFlipX。站立帧 = 跑步 GIF 的第 0 帧（静止）。与椅子交互（E键）时角色坐下，按任意方向键起身。坐下时深度排序由椅子Y坐标决定。

```typescript
// client/src/game/systems/PlayerController.ts
import Phaser from "phaser";

type Direction = "left" | "right" | "up" | "down";

export class PlayerController {
  private wasd: Record<string, Phaser.Input.Keyboard.Key>;
  private player: Phaser.GameObjects.Sprite;
  private body: Phaser.Physics.Arcade.Body;
  private speed = 120;
  private frozen = false;
  private lastDirection: Direction = "down"; // 默认朝下（front）

  constructor(scene: Phaser.Scene, player: Phaser.GameObjects.Sprite) {
    this.player = player;
    this.body = player.body as Phaser.Physics.Arcade.Body;
    this.wasd = {
      W: scene.input.keyboard!.addKey(Phaser.Input.Keyboard.KeyCodes.W),
      A: scene.input.keyboard!.addKey(Phaser.Input.Keyboard.KeyCodes.A),
      S: scene.input.keyboard!.addKey(Phaser.Input.Keyboard.KeyCodes.S),
      D: scene.input.keyboard!.addKey(Phaser.Input.Keyboard.KeyCodes.D),
    };
  }

  update(_delta: number) {
    // 坐下时检测方向键 → 站起来
    if (this.isSitting) {
      if (
        Phaser.Input.Keyboard.JustDown(this.wasd.W) ||
        Phaser.Input.Keyboard.JustDown(this.wasd.A) ||
        Phaser.Input.Keyboard.JustDown(this.wasd.S) ||
        Phaser.Input.Keyboard.JustDown(this.wasd.D)
      ) { this.standUp(); }
      this.body.setVelocity(0, 0);
      return;
    }

    if (this.frozen) { this.body.setVelocity(0, 0); return; }

    let vx = 0, vy = 0;
    let currentDir: Direction | null = null;

    // A=左, D=右（水平方向优先）
    if (this.wasd.A.isDown)      { vx = -1; currentDir = "left"; }
    else if (this.wasd.D.isDown) { vx = 1;  currentDir = "right"; }

    // W=上, S=下
    if (this.wasd.W.isDown)      { vy = -1; currentDir = "up"; }
    else if (this.wasd.S.isDown) { vy = 1;  currentDir = "down"; }

    // 对角线归一化（水平+垂直同时按时，保留两个方向的速度）
    if (vx && vy) { vx *= 0.7071; vy *= 0.7071; }

    this.body.setVelocity(vx * this.speed, vy * this.speed);

    if (currentDir) {
      // 移动中：记录最后朝向 + 播放对应方向的跑步动画
      this.lastDirection = currentDir;
      this.playRunAnimation(currentDir);
    } else {
      // 停下：播放最后朝向的站立动画
      this.playIdleAnimation(this.lastDirection);
    }
  }

  /** 播放跑步动画（6帧GIF循环） */
  private playRunAnimation(dir: Direction) {
    switch (dir) {
      case "left":
        this.player.play("yps_run_left", true);
        break;
      case "right":
        this.player.play("yps_run_right", true);   // 独立GIF，无需翻转
        break;
      case "up":
        this.player.play("yps_run_back", true);    // 背面 = 向上
        break;
      case "down":
        this.player.play("yps_run_front", true);   // 正面 = 向下
        break;
    }
    this.player.setFlipX(false);  // GIF方案不需要翻转
  }

  /** 播放站立动画（GIF第0帧，静止） */
  private playIdleAnimation(dir: Direction) {
    // 站立帧 = 对应方向GIF的第0帧，设置为暂停状态
    switch (dir) {
      case "left":
        this.player.play("yps_idle_left", true);
        break;
      case "right":
        this.player.play("yps_idle_right", true);
        break;
      case "up":
        this.player.play("yps_idle_back", true);
        break;
      case "down":
        this.player.play("yps_idle_front", true);
        break;
    }
    this.player.setFlipX(false);
  }

  /** 播放坐下动画（sit静止帧） */
  private playSitAnimation(dir: Direction) {
    // 坐下使用静止帧（视觉上与站立相同方向但姿势不同，如有sit专用帧则替换）
    this.playIdleAnimation(dir);
  }

  /**
   * 让角色坐下（与椅子交互时调用）
   * @param chairY 椅子底部 Y 坐标，用于前后遮挡
   * @param sitInFront true=坐在椅子前面，false=坐在后面
   */
  sit(chairY: number, sitInFront: boolean) {
    this.isSitting = true;
    this.body.setVelocity(0, 0);
    this.playSitAnimation(this.lastDirection);
    // 深度排序：前面时深度高于椅子，后面时低于椅子
    this.player.setDepth(sitInFront ? (chairY + 1) / 768 : (chairY - 1) / 768);
  }

  /** 站起来 */
  standUp() {
    if (!this.isSitting) return;
    this.isSitting = false;
    this.playIdleAnimation(this.lastDirection);
  }

  freeze() { this.frozen = true; }
  unfreeze() { this.frozen = false; }
  getPosition() { return { x: this.player.x, y: this.player.y }; }
}
```

### 7.2 InteractionSystem

```typescript
// client/src/game/systems/InteractionSystem.ts
import Phaser from "phaser";
import { gameBridge } from "../bridge/GameBridge";

interface Interactable {
  x: number;
  y: number;
  type: "npc" | "item" | "door";
  id: string;
  sceneId?: string;
  targetMap?: string;
  spawnId?: string;
}

export class InteractionSystem {
  private scene: Phaser.Scene;
  private player: Phaser.GameObjects.Sprite;
  private interactables: Interactable[] = [];
  private interactKey: Phaser.Input.Keyboard.Key;
  private interactRadius = 48;
  private interactHint: Phaser.GameObjects.Text | null = null;

  constructor(scene: Phaser.Scene, player: Phaser.GameObjects.Sprite) {
    this.scene = scene;
    this.player = player;
    this.interactKey = scene.input.keyboard!.addKey(Phaser.Input.Keyboard.KeyCodes.E);
  }

  registerInteractable(obj: Interactable) {
    this.interactables.push(obj);
  }

  update() {
    // 找最近的可交互对象
    let nearest: Interactable | null = null;
    let minDist = this.interactRadius;

    for (const obj of this.interactables) {
      const dist = Phaser.Math.Distance.Between(
        this.player.x, this.player.y, obj.x, obj.y
      );
      if (dist < minDist) { minDist = dist; nearest = obj; }
    }

    // 显示/隐藏交互提示
    if (nearest) {
      if (!this.interactHint) {
        this.interactHint = this.scene.add.text(0, 0, "[E] 交互", {
          fontSize: "12px", color: "#ffff00",
          backgroundColor: "#00000088", padding: { x: 4, y: 2 },
        }).setDepth(1).setOrigin(0.5);
      }
      this.interactHint.setPosition(nearest.x, nearest.y - 30);
      this.interactHint.setVisible(true);
    } else {
      this.interactHint?.setVisible(false);
    }

    // E 键触发
    if (Phaser.Input.Keyboard.JustDown(this.interactKey) && nearest) {
      switch (nearest.type) {
        case "npc":
          gameBridge.sendToReact({
            type: "TRIGGER_DIALOGUE",
            sceneId: nearest.sceneId || nearest.id,
            mapId: this.scene.registry.get("currentMapId") || "",
          });
          break;
        case "item":
          gameBridge.sendToReact({
            type: "TRIGGER_ITEM",
            itemId: nearest.id,
            mapId: this.scene.registry.get("currentMapId") || "",
          });
          break;
        case "door":
          gameBridge.sendToReact({
            type: "TRIGGER_DOOR",
            targetMap: nearest.targetMap || "",
            spawnId: nearest.spawnId || "",
          });
          break;
      }
    }
  }
}

```typescript
// client/src/game/systems/TriggerSystem.ts
import Phaser from "phaser";
import { gameBridge } from "../bridge/GameBridge";

interface TriggerConfig {
  type: string;        // "dialogue" | "door" | "effect"
  sceneId?: string;
  targetMap?: string;
  spawnId?: string;
  requireFlag?: string;
  setFlag?: string;
}

interface Trigger {
  zone: Phaser.GameObjects.Zone;
  config: TriggerConfig;
  triggered: boolean;
}

export class TriggerSystem {
  private scene: Phaser.Scene;
  private player: Phaser.GameObjects.Sprite;
  private triggers: Trigger[] = [];

  constructor(scene: Phaser.Scene, player: Phaser.GameObjects.Sprite) {
    this.scene = scene;
    this.player = player;
  }

  registerTrigger(zone: Phaser.GameObjects.Zone, config: TriggerConfig) {
    this.triggers.push({ zone, config, triggered: false });
  }

  update() {
    for (const trigger of this.triggers) {
      if (trigger.triggered) continue;

      // 检查前置条件
      if (trigger.config.requireFlag) {
        const flags = gameBridge.gameState?.flags || {};
        if (!flags[trigger.config.requireFlag]) continue;
      }

      // 检查玩家是否在触发区内
      const zone = trigger.zone;
      const body = zone.body as Phaser.Physics.Arcade.StaticBody;
      if (
        this.player.x >= body.x &&
        this.player.x <= body.x + body.width &&
        this.player.y >= body.y &&
        this.player.y <= body.y + body.height
      ) {
        trigger.triggered = true;

        // 设置 flag
        if (trigger.config.setFlag) {
          // 通过 bridge 通知 React 设 flag
        }

        switch (trigger.config.type) {
          case "dialogue":
            gameBridge.sendToReact({
              type: "TRIGGER_DIALOGUE",
              sceneId: trigger.config.sceneId || "",
              mapId: this.scene.registry.get("currentMapId") || "",
            });
            break;
          case "door":
            gameBridge.sendToReact({
              type: "TRIGGER_DOOR",
              targetMap: trigger.config.targetMap || "",
              spawnId: trigger.config.spawnId || "",
            });
            break;
        }
      }
    }
  }
}

// client/src/game/config/animationRegistry.ts
import Phaser from "phaser";

/**
 * 角色动作方案（v8.0 / GIF序列帧）
 * 
 * ───────────────────────────────────────────────────────────────────────
 * 文件结构（每个角色，每个方向，独立 GIF）
 * ───────────────────────────────────────────────────────────────────────
 * 
 *   角色：yps（叶平生）、ly（刘宇）
 *   方向：front(正面/向下)  back(背面/向上)  left(向左)  right(向左镜像)
 * 
 *   yps_frames/
 *     yps_frames_front.gif   32×64px  6帧  向下跑步
 *     yps_frames_back.gif    32×64px  6帧  向上跑步
 *     yps_frames_left.gif    32×64px  6帧  向左跑步
 *     yps_frames_right.gif   32×64px  6帧  向右跑步（由left镜像生成）
 * 
 *   ly_frames/
 *     ly_frames_front.gif    32×64px  6帧  向下跑步
 *     ly_frames_back.gif     32×64px  6帧  向上跑步
 *     ly_frames_left.gif     32×64px  6帧  向左跑步
 *     ly_frames_right.gif    32×64px  6帧  向右跑步（由left镜像生成）
 * 
 * ───────────────────────────────────────────────────────────────────────
 * Phaser 加载 GIF 的两种方案（选其一）
 * ───────────────────────────────────────────────────────────────────────
 * 
 * ✅ 方案A（推荐）：预转 spritesheet + 直接用 Phaser 原生 spritesheet
 *   - 用 Python/Node 将 GIF 转为水平排列的 spritesheet PNG（192×64px，6帧横排）
 *   - 用 this.load.spritesheet() 加载，frameWidth=32, frameHeight=64
 *   - 优点：无额外插件，Phaser 原生支持
 * 
 * ⚠️ 方案B：使用 rexGifPlugin（rex-plugins）
 *   - 需安装 phaser3-rex-plugins
 *   - this.load.gif(key, path)
 *   - 优点：直接加载 GIF，无需预转
 *   - 缺点：多一个依赖，GIF 首帧加载有延迟
 * 
 * 本文档采用 **方案A**，即将 GIF 预转为 spritesheet PNG 再加载。
 * 预转脚本：scripts/gif-to-spritesheet.py（每个 GIF → 同名 .png）
 * 
 * ───────────────────────────────────────────────────────────────────────
 * 动画名称约定
 * ───────────────────────────────────────────────────────────────────────
 * 
 *   {角色}_run_{方向}   跑步动画（帧0~5，循环，frameRate=10）
 *   {角色}_idle_{方向}  站立静止（帧0，不循环）
 * 
 *   方向：front | back | left | right
 * 
 *   示例：
 *     yps_run_front    yps_run_back    yps_run_left    yps_run_right
 *     yps_idle_front   yps_idle_back   yps_idle_left   yps_idle_right
 *     ly_run_front     ly_run_back     ly_run_left     ly_run_right
 *     ly_idle_front    ly_idle_back    ly_idle_left    ly_idle_right
 * 
 * ───────────────────────────────────────────────────────────────────────
 * 对应 PlayerController 的 Direction 类型
 * ───────────────────────────────────────────────────────────────────────
 * 
 *   WASD → Direction:  W→"up"(back)  S→"down"(front)  A→"left"  D→"right"
 *   Direction 到动画名的映射：
 *     "up"    → yps_run_back / yps_idle_back
 *     "down"  → yps_run_front / yps_idle_front
 *     "left"  → yps_run_left / yps_idle_left
 *     "right" → yps_run_right / yps_idle_right
 */

/** 将 GIF 转 spritesheet 后的图片路径 */
const SPRITE_PATHS = {
  yps: {
    front: "/assets/sprites/yps_frames/yps_frames_front.png",
    back:  "/assets/sprites/yps_frames/yps_frames_back.png",
    left:  "/assets/sprites/yps_frames/yps_frames_left.png",
    right: "/assets/sprites/yps_frames/yps_frames_right.png",
  },
  ly: {
    front: "/assets/sprites/ly_frames/ly_frames_front.png",
    back:  "/assets/sprites/ly_frames/ly_frames_back.png",
    left:  "/assets/sprites/ly_frames/ly_frames_left.png",
    right: "/assets/sprites/ly_frames/ly_frames_right.png",
  },
};

/** 每个 spritesheet：192×64px，6帧横排（frameWidth=32, frameHeight=64） */
const FRAME_CONFIG = { frameWidth: 32, frameHeight: 64 };

/**
 * 在 PreloadScene.preload() 中调用
 */
export function preloadSpriteSheets(scene: Phaser.Scene) {
  for (const [charId, dirs] of Object.entries(SPRITE_PATHS)) {
    for (const [dir, path] of Object.entries(dirs)) {
      scene.load.spritesheet(`${charId}_${dir}`, path, FRAME_CONFIG);
    }
  }
}

/**
 * 在 PreloadScene.create() 中调用
 * 注册所有角色的四方向跑步+站立动画
 */
export function createPlayerAnimations(scene: Phaser.Scene) {
  for (const charId of ["yps", "ly"]) {
    for (const dir of ["front", "back", "left", "right"]) {
      const key = `${charId}_${dir}`;   // spritesheet key

      // 跑步：帧 0~5，循环，frameRate=10
      scene.anims.create({
        key: `${charId}_run_${dir}`,
        frames: scene.anims.generateFrameNumbers(key, { start: 0, end: 5 }),
        frameRate: 10,
        repeat: -1,
      });

      // 站立：帧 0，不循环（跑步序列第1帧作为站立帧）
      scene.anims.create({
        key: `${charId}_idle_${dir}`,
        frames: [{ key, frame: 0 }],
        frameRate: 1,
        repeat: 0,
      });
    }
  }
}
```

> **GIF → spritesheet PNG 转换脚本（方案A必需）**
> 
> 在 `scripts/gif-to-spritesheet.py` 中运行：
> 
> ```python
> from PIL import Image
> import os, glob
> 
> DIRS = [
>     "client/public/assets/sprites/yps_frames",
>     "client/public/assets/sprites/ly_frames",
> ]
> 
> for folder in DIRS:
>     for gif_path in glob.glob(f"{folder}/*.gif"):
>         img = Image.open(gif_path)
>         frames = []
>         try:
>             while True:
>                 frames.append(img.copy().convert("RGBA"))
>                 img.seek(img.tell() + 1)
>         except EOFError:
>             pass
>         
>         # 水平拼接为 spritesheet（192×64px，6帧横排）
>         w, h = frames[0].size   # 32×64
>         sheet = Image.new("RGBA", (w * len(frames), h))
>         for i, f in enumerate(frames):
>             sheet.paste(f, (i * w, 0))
>         
>         out_path = gif_path.replace(".gif", ".png")
>         sheet.save(out_path)
>         print(f"  {os.path.basename(gif_path)} → {os.path.basename(out_path)} ({w*len(frames)}×{h})")
> ```
> 
> 每次修改 GIF 源文件后需重新运行此脚本。

```typescript
import { useEffect, useRef } from "react";
import Phaser from "phaser";
import { PhaserGameConfig } from "../game/config/PhaserGameConfig";
import { gameBridge } from "../game/bridge/GameBridge";
import type { GameAction } from "../engine/gameReducer";

interface Props {
  dispatch: React.Dispatch<GameAction>;
  currentMapId: string;
  onDialogueTrigger: (sceneId: string) => void;
}

export function PhaserContainer({ dispatch, currentMapId, onDialogueTrigger }: Props) {
  const gameRef = useRef<Phaser.Game | null>(null);
  const containerRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    if (!containerRef.current || gameRef.current) return;

    const config = PhaserGameConfig(containerRef.current);
    gameRef.current = new Phaser.Game(config);

    // 监听 Phaser 触发对话 → 打开 DialogOverlay
    gameBridge.onPhaserEvent("TRIGGER_DIALOGUE", (event) => {
      gameBridge.sendToPhaser({ type: "FREEZE_PLAYER" });
      onDialogueTrigger(event.sceneId);
    });

    // 监听门触发 → 切换地图
    gameBridge.onPhaserEvent("TRIGGER_DOOR", (event) => {
      dispatch({
        type: "CHANGE_MAP",
        mapId: event.targetMap,
        spawnId: event.spawnId,
        position: { x: 0, y: 0 },
      });
      gameBridge.sendToPhaser({
        type: "CHANGE_MAP",
        mapId: event.targetMap,
        spawnId: event.spawnId,
      });
    });

    return () => {
      gameRef.current?.destroy(true);
      gameRef.current = null;
      gameBridge.removeAllListeners();
    };
  }, []);

  return (
    <div
      ref={containerRef}
      style={{
        width: "100%",
        height: "100%",
        position: "absolute",
        top: 0, left: 0,
      }}
    />
  );
}

## 10. 大地图设计规范

### 10.1 核心原则：整张长图 + 镜头跟随，无需拼接

对于走廊、操场等比视口大的场景，**不采用分区拼接**，而是绘制一整张大尺寸底图，利用 Phaser 相机的 `startFollow` 实现镜头跟随。

**原因**：
- Phaser 相机原生支持大地图滚动，代码量几乎为零
- 玩家体验最好——无接缝、无加载中断
- 美术只需画一张图，无需对齐接缝
- 本项目所有场景均 < 8000px，不存在加载性能瓶颈

**已有代码**（`MapScene.ts` 第 465-467 行，无需修改）：

```typescript
this.physics.world.setBounds(0, 0, this.map.widthInPixels, this.map.heightInPixels);
this.cameras.main.setBounds(0, 0, this.map.widthInPixels, this.map.heightInPixels);
this.cameras.main.startFollow(this.player, true, 0.08, 0.08);
```

- `setBounds`：根据地图实际像素尺寸设置世界边界和相机边界
- `startFollow`：相机始终跟随玩家，`true` 表示启用 lerp 平滑插值，`0.08` 为插值系数
- 不管地图是 1407×768 还是 4000×800，这两行代码都自动适配

### 10.2 地图分类与尺寸建议

根据场景规模，将地图分为三类：

### 10.6 相机跟随参数调优

`startFollow` 的第三个和第四个参数控制镜头平滑度：

```typescript
// 当前设置（已在 MapScene.ts 中）
this.cameras.main.startFollow(this.player, true, 0.08, 0.08);
//                                                    ↑      ↑
//                                              roundPx  lerpX  lerpY
```

| 参数 | 值 | 效果 |
|------|---|------|
| lerpX/lerpY = 1.0 | 即时跟随 | 镜头严格跟玩家，无延迟 |
| lerpX/lerpY = 0.08 | 缓动跟随 | 镜头平滑追赶，有电影感 |
| lerpX/lerpY = 0.01 | 极缓跟随 | 镜头缓慢漂移，适合大场景 |

**建议**：
- 小房间（卫生间、卧室）：`lerp = 1.0`，即时跟随
- 中等场景（客厅、教室）：`lerp = 0.08`，当前值即可
- 大地图（走廊、操场）：`lerp = 0.05~0.08`，保持当前值，或根据手感微调

如需按场景动态调整，可在 `loadMap` 中根据 `MapRegistry[mapId].width` 判断：

```typescript
const lerp = entry.width > 2000 ? 0.05 : 0.08;
this.cameras.main.startFollow(this.player, true, lerp, lerp);
```

### 10.7 何时需要分区？

**分区 = 把一个场景拆成多张 mapId，边界处用 door trigger 切换。**

仅当以下条件满足时才考虑分区：

| 条件 | 阈值 | 原因 |
|------|------|------|
| 底图尺寸超过 8000px | 单张图加载超过 2s | 影响首屏体验 |
| 逻辑上是完全不同的区域 | 如走廊 vs 教室 | 不同区域用不同 mapId 更清晰 |
| 底图风格/图集完全不同 | 如室内 vs 室外 | 无法共用 Image Layer |

**本项目中，所有场景均不超过 4000px，无需分区。** 走廊、操场各自画一张整图即可。

### 10.8 大地图与地图切换的关系

大地图内部不分区，但大地图**之间**通过 door trigger 切换。切换流程（已有代码）：

```
走廊(corridor) ──door──→ 教室(classroom)     ← 小地图
走廊(corridor) ──door──→ 画廊(wang_gallery)   ← 小地图
操场(playground_gate) ──door──→ 教学楼入口     ← 另一张大地图
```

切换时 `MapScene.transitionToMap()` 会：
1. `fadeOut(300ms)` 淡出

## 11.5 多边形遮挡（Polygon Occlusion）

### 背景

对于形状不规则的物品（如斜放的家具、异形装饰等），矩形的 sprite 叠层方案无法精准勾勒遮挡轮廓。多边形遮挡提供了一种更精确的方式：

- 在 `furniture_objects` 对象层中定义多边形对象（含 `polygon` 顶点数组）
- 运行时从底图上裁剪多边形对应区域渲染（而非纯色填充），视觉上呈现"底图被提起"的效果
- 同时支持叠加精灵图纹理（`spriteKey` 属性），精灵图在裁剪底图之上显示
- 深度（depth）按多边形底部 Y 坐标计算，与玩家 Y 轴排序一致
- 碰撞用包围盒矩形近似（Arcade Physics 限制）

### Tiled JSON 中的 polygon 格式

```json
{
  "name": "sofa_oblique",
  "type": "furniture",
  "x": 400,
  "y": 300,
  "width": 0,
  "height": 0,
  "polygon": [
    { "x": 0,   "y": 0   },
    { "x": 120, "y": 10  },
    { "x": 115, "y": 80  },
    { "x": 5,   "y": 85  }
  ],
  "properties": [
    { "name": "walkable", "value": false, "type": "bool" },
    { "name": "spriteKey", "value": "sofa", "type": "string" }
  ]
}
```

关键点：
- `polygon` 数组包含 ≥3 个顶点，坐标相对于对象的 `(x, y)`
- `width` 和 `height` 设为 0（多边形自动计算包围盒）
- 可选 `walkable` 属性控制是否产生碰撞
- 可选 `spriteKey` 属性指定叠加的精灵图纹理

### MapScene 中的处理逻辑

`MapScene.ts` 的 `furniture_objects` 处理循环中，在精灵图逻辑之前检查 `obj.polygon`：

1. 若对象有 `polygon` 字段 → 裁剪底图对应区域 + 叠加精灵图 → `continue` 跳过精灵图逻辑
2. 否则走原有的精灵图叠层逻辑（向后兼容）

多边形遮挡物：
- 内容：从底图裁剪的多边形区域（clip + drawImage），并可选叠加精灵图
- 深度：`(obj.y + maxY) / mapHeight`，与玩家 Y 排序一致
- 碰撞：用包围盒矩形（minX/minY → maxX/maxY）
- 纹理 key：`poly_{obj.id}_{timestamp}_{random}` 确保唯一性，避免同名对象冲突

**已知问题与修复**：

| 问题 | 原因 | 修复 |
|------|------|------|
| Texture key already in use | 多个同名多边形对象 + Date.now() 精度不足 | key 改用 obj.id + random 后缀 |
| 碰撞框拖拽预览与实际不一致 | Math.max(32, rawW) 强制最小值 | 去掉强制最小值 |
| 多边形顶点拖拽误触 | 无最小移动阈值 | 添加 3px 阈值 |
| balcony_night 缓存未命中 | 与 balcony 共享 map.json，Phaser 去重 | MapScene 中添加 fallback 从 map_balcony 深拷贝 |

### 与精灵图方案的关系

| 方案 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| 精灵图叠层 | 矩形物品（dormitory 等） | 显示物品纹理，视觉无缝 | 只能矩形 |
| 多边形遮挡 | 不规则形状物品 | 精准勾勒遮挡轮廓 + 底图裁剪 + 可选精灵图 | 实现略复杂 |

两种方案在 `furniture_objects` 层中共存，根据对象是否包含 `polygon` 字段自动区分。

> **提示**：使用 `scripts/map_editor.html` 工具可直接在底图上点击绘制多边形，无需手动编写 JSON 坐标。编辑器支持 Ctrl+Z 撤销顶点、拖拽顶点微调、叠加精灵图预览等功能。详细指南见 `docs/08_地图编辑器使用指南.md`。
