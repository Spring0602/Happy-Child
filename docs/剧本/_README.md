# 剧本目录

> 每个 `.md` 文件按 `剧本编码格式规范_v1.0.md` 编写。
> 我读取后自动翻译为 `client/src/data/scenes.ts` 代码。

## 文件组织

| 文件 | 内容 |
|------|------|
| `_npc设定.md` | 全局 NPC 角色设定卡片（所有章节共用） |
| `ch1_前兆_宿舍.md` | 第1章 · 宿舍场景（凌晨04:46-次日清晨） |
| `ch2_副本开启.md` | 第2章 · 技能抽取、家庭规则 |
| `ch3_学校初入.md` | 第3章 · 刘宇登场、空座位 |
| ... | ... |

## 工作流

```
你写                       我编译
┌────────────────┐      ┌──────────────────┐
│ docs/剧本/       │      │ client/src/data/   │
│ ch1_前兆_宿舍.md  │ ───→ │ scenes.ts         │
│ ch2_副本开启.md   │ 我读  │ (追加/更新场景)    │
│ _npc设定.md      │      │ characters.ts     │
└────────────────┘      └──────────────────┘
```

1. 你在 `.md` 里按编码格式写剧本
2. 我读取后增量更新 `scenes.ts`、`characters.ts` 等
3. 不改动你的 `.md` 之外的内容

## 编写顺序

按章节推进，写完一章再开下一章。`_npc设定.md` 在新 NPC 首次出场前补写即可。
