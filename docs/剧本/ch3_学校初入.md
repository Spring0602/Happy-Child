# 第3章 · 学校初入

> 本章核心：主角第一次进入学校区域，在“似曾相识的高中班级”中确认自己身份错位，并与刘宇建立最初的有限合作关系。
>
> 固定主线：
> - 主角在晚间教室观察到班级气氛与自己高中记忆高度重合。
> - 刘宇恶作剧藏周骐瑞作业，主角被卷入并第一次真正放松发笑。
> - 主角发现记忆中的自己所在座位空着，从而意识到副本身份存在错位。
> - 刘宇察觉主角异常，并主动替主角解围。
> - 晚间测试实为“优秀学生品质测试”，主角确认自己试卷与普通学生不同。
> - 校门口，主角利用“家里出事”的说法诱导刘宇承诺帮忙，并确认班级人数为39人。
> - 回家后，计划本出现“我”的红笔批注，主角因负面思想触发强烈违规提醒。
> - 主角用白天在学校获得的真实快乐与被关心经验，说服“我”暂时活下去。
>
> 动态叙事边界：
> - 玩家可选择课堂中如何应对周骐瑞追问、如何向刘宇释放求助信号、如何解释校门口的异常。
> - 玩家不能提前知道学校规则，不能通过本章调查到花名册，也不能直接识破空座位对应的真实身份。
> - 刘宇可以表现出熟悉、敏锐和讲义气，但不能承认自己知道副本真相，不能主动泄露学校规则。
> - 周骐瑞可以表现出担忧与理性，但不能在本章成为可合作对象。
