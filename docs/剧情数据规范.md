# 剧情数据规范

剧情数据主要写在：

- `client/src/data/scenes.ts`
- `client/src/data/characters.ts`
- `client/src/data/endings.ts`
- `client/src/data/rules.ts`

## 1. Scene 剧情节点

```ts
{
  id: "unique_scene_id",
  chapter: "章节名称",
  background: "/assets/bg/xxx.svg",  // ⚠️ 已弃用 — 场景改用 Phaser 像素地图
  speaker: "说话人",                // 决定立绘显示位置
  character: "角色ID，可选",
  text: "显示给玩家的剧情文本",
  choices: [/* 选项，可选 */],
  nextSceneId: "下一节点ID，可选",
  aiEvent: "npc_dialogue 或 ending_judge，可选"
}
```

规则：

- `id` 必须唯一。
- 普通剧情节点使用 `nextSceneId`。
- 选择节点使用 `choices`。
- AI 事件节点使用 `aiEvent`。
- 不要同时写太多逻辑在一个节点里，每个节点最好只承担一个叙事动作。
- `background` 字段已弃用（不再使用 SVG 背景图），场景由 Phaser 实时渲染当前所在区域的像素地图。
- `speaker` 字段决定 DialogOverlay 中的立绘位置：
  - 叶平生（主角）→ 左侧立绘
  - 刘宇、王老师等 NPC → 右侧立绘
  - 旁白/系统/规则 → 无立绘（纯文本框）

## 2. Choice 选项

```ts
{
  id: "choice_id",
  text: "玩家看到的选项文本",
  nextSceneId: "选择后跳转的剧情节点",
  effects: {
    // 认知层：玩家是否看见了真实自我与世界
    truthDesire: 2,        // 真相欲望：是否愿意追问副本本质
    joyPerception: 1,       // 快乐感知：是否能从功利评价之外感受生命
    
    // 行动层：认知之后，玩家是否付出实际行动
    authorityResistance: 1, // 权威抵制度：是否敢质疑系统；区分"无脑叛逆"与"建设性叛逆"
    realityJudgment: -1,    // 现实判断：是否知道什么时候退让、什么时候冒险
    selfProtection: 1,      // 自我保护：是否保留筹码；但是否因过度保护而无所作为
    
    // 影响层：玩家是否尝试影响周围的人/世界
    empathy: 1,             // 共情能力：是否理解NPC、"我"和普通学生的痛苦
    trust: -1               // 关系信任：是否愿意与他人合作；是否相信他人有能力改变
  },
  tags: ["理性", "风险"],
  needAIAnalysis: true
}
```

设计原则：

- 不做简单善恶选项。
- 每个选项都应有道理，也有代价。
- 选项主要改变人格画像，而不是直接决定结局。
- **认知层指标**（truthDesire, joyPerception）：衡量玩家是否看见了真实自我与世界；认同"我"只是第一步。
- **行动层指标**（authorityResistance, realityJudgment, selfProtection）：衡量玩家在认知之后是否付出行动；区分"破坏性叛逆"与"建设性叛逆"；做出成果才是实在的。
- **影响层指标**（empathy, trust）：衡量玩家是否尝试影响周围的人/世界；真正的进化是打破丛林法则，为世界带来光。

## 3. 推荐剧情节点拆分

后续补第1-8章时，可按以下节点添加：

1. `prologue_bug_fixed`：凌晨修完 bug。
2. `computer_reopen`：电脑异常重启。
3. `anonymous_email`：收到人类进化计划邮件。
4. `store_meet_players`：厨房用品店遇到其他参赛者。
5. `instance_start`：进入候场区。
6. `skill_awaken`：获得权威抵制与规则破坏者身份。
7. `home_planbook`：发现计划表。
8. `family_food_test`：早餐与食物风险。
9. `bathroom_mirror_rule`：卫生间镜子规则。
10. `school_exam`：优秀学生品质测试。
11. `liuyu_first_trust`：刘宇试探与合作。
12. `art_class`：王老师美术课。
13. `school_food_solution`：倒掉饭菜。
14. `wang_teacher_trade`：镜中尸骸、湖中遗物、书中落叶。
15. `class_three_escape`：3班逃生。
16. `teacher_punishment`：办公室怪物惩罚。
17. `school_chase`：全校追杀。
18. `delete_good_child`：删除“好孩子”。
19. `mirror_truth_1`：镜中真相碎片。
20. `rooftop_dialogue`：天台与“我”的对话。
