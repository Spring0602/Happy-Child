# 开发路线图

## Demo 定位（已确认）

- **Demo内容**：第1-8章，主角叶平生与内在"我"达成和解的认知弧。
- **评判维度**：采用完整哲学框架（认知→行动→影响三层），不因Demo只做认知弧而简化。
- **长期目标**：比赛结束后继续完成整个副本（当前为1/3），作为个人作品集。
- **核心创新**：玩家选择→AI综合分析七维人格画像→动态匹配最符合"事物自然规律"的结局。

## 第二阶段：剧情扩展（认知弧） ✅ 已完成

目标：把第1-8章改编成可玩节点，聚焦"和解"认知弧。

**核心场景序列**（Demo闭环）：
1. `prologue_email`：收到邮件，展示玩家风险判断。
2. `home_planbook`：家庭规则，理解"快乐"被强制。
3. `school_exam`：学校初入，优秀学生品质测试。
4. `liuyu_meet`：刘宇登场，测试玩家是否值得合作。
5. `art_class`：美术课，王老师提示"做自己"。
6. `wang_trade`：王老师交易，三个谜题。
7. `ritual_gong`：功成名就祈愿仪式（核心展示场景）。
8. `rooftop_dialogue`：天台与"我"的对话（Demo核心选择点）。
9. `ending_judge`：AI裁决，展示人格画像与结局。

**评判维度**：采用完整七维人格框架（认知→行动→影响三层），不因Demo只做认知弧而简化。

- [x] 序章：收到邮件。
- [x] 家庭区域：计划表、食物、镜子规则。
- [x] 学校区域：考试、刘宇、空座位。
- [x] 美术课：王老师"做自己"。
- [x] 王老师交易：三个谜题。
- [x] 3班逃生。
- [x] 全校追杀。
- [x] 删除"好孩子"。
- [x] 天台对话。
